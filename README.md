# Linux System Health Monitor 🐧

Простой Bash-скрипт для автоматического мониторинга ресурсов системы. Проект создан для демонстрации навыков работы с терминалом Linux, автоматизации задач и написания shell-скриптов.

## 🎯 Цели проекта
- Автоматизировать сбор данных о состоянии сервера (CPU, RAM, Disk).
- Организовать логирование данных в текстовый файл.
- Использовать стандартные утилиты Linux (Coreutils) для системного администрирования.

## 🛠 Технологический стек
* **Shell**: Bash
* **Инструменты**: `df`, `free`, `ps`, `grep`, `tee`
* **Автоматизация**: Cron (планировщик задач)

## 📋 Функционал скрипта
Скрипт `monitor.sh` выполняет следующие действия:
1. **Disk Usage**: Проверяет свободное место на разделах диска.
2. **Memory Usage**: Выводит объем занятой и свободной оперативной памяти в Мб.
3. **Process Tracking**: Определяет ТОП-5 процессов, наиболее нагружающих процессор.
4. **Logging**: Сохраняет результат в файл `system_report.log` с указанием точного времени запуска.
